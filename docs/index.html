How the Simulation Works

Each simulation run plays out all unplayed group stage matches by drawing scores from a Poisson distribution, where the number of goals per team per match is modeled by team-specific FIFA ranking points. Group standings are then determined using official FIFA 2026 tiebreaker rules, including head-to-head points, goal difference, and goals scored among tied teams. This process is repeated 10,000 times, and the results are aggregated into the probabilities shown on this dashboard.
